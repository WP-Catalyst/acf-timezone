An Advanced Custom Fields plugin that allows you to pick from a list of Timezones.

== Description ==

Timezone Field for Advanced Custom Fields is an easy-to-use WordPress plugin that extends Advanced Custom Fields (Free or Pro) to
allow you to add a timezone select field to your field groups. Whether you're managing a news site 
or planning an event, this plugin is the perfect tool to simplify your workflow and save you time.

## Compatibility

This ACF field type is compatible with:

* ACF 6

== Installation ==

### INSTALL FROM WITHIN WORDPRESS

1. Visit the plugins page within your dashboard and select ‘Add New’.
1. Search for ‘Timezone Field for Advanced Custom Fields’.
1. Activate Timezone Field for Advanced Custom Fields from your Plugins page.

### INSTALL MANUALLY

1. Upload the ‘wpc-acf-timezone’ folder to the /wp-content/plugins/ directory.
1. Activate the Timezone Field for Advanced Custom Fields plugin through the ‘Plugins’ menu in WordPress.
